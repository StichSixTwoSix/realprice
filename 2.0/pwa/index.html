О пректе

Этот проект создан по мотивам поста на Пикабу

Здесь можно рассчитать честную цену за килограмм, литр или штуку. Для того, чтобы узнать цену, нужно ввести вес (или объем для жидкостей) упаковки и её цену. Так же можно расчитать стоимость упаковки по весу и цене за килограмм.

При нажатии на кнопку "Рассчитать" производится расчет и результат выводится в поле. Так же, результат дублируется в журнале внизу для удобного сравнения. Очистить журнал можно кнопкой "Очистка".

Калькулятор сделан максимально простым, без всего лишнего. Тут нет рекламы, платных подписок, ограничений по количеству ввода и размера журнала. Журнал не сохраняется, можно делать скриншоты.

Чтобы видеть, нужно ли это кому-то, подключен счетчик от mail.ru. Это сделано исключительно для общей статистики. Всё, что связано с куками - это их рук дело.

Приложения

По ссылкам ниже можно скачать приложения для Android на Google play и RuStore.

Для пользователей iPhone и других устройств есть PWA версия, которая настолько универсальная, что подходит практически для любого устройства, в том числе и для устройств на базе Windows, Linux, Android. Инструкцию по установке можно посмотреть в посте на Pikabu. PWA приложение - полная копия, за исключением счетчика от mail.ru.
